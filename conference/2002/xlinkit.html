Demonstration title
Integrating Multiple Views With xlinkit
Organizers
  • Christian Nentwich (University College London, GB)
  • Wolfgang Emmerich (University College London, GB)
  • Anthony Finkelstein (University College London, GB)
Day
Thursday April 25, 4:45-5:30
Friday April 26, 11:45-12:30
Abstract

We will demonstrate how xlinkit, a generic technology for checking the consistency of distributed, heterogeneous documents, can be applied to checking the consistency of multiple-view software development artifacts at different stages of the lifecycle.

In particular, we will concentrate on demonstrating the consistency relationships that arise in Enterprise JavaBeans-based systems. We will show to how to express and check simple constraints between UML designs, Java implementations and deployment descriptors.

The checking demo will be followed by a demonstration of xlinkit's report generation toolkit that allows the generation of human readable, HTML formatted error reports containing diagnostic information about inter-view inconsistencies.

More information about xlinkit can be found at http://www.xlinkit.com, including an evaluation package, several introductory on-line examples, and an XMI file checker.

Tutorials

For this first full conference on AOSD we received a considerable number of high-quality tutorial submissions. We aimed for an optimal program that covers the state of the art in AOSD, appealing to participants ranging from novice to expert in this field, and with limited overlap between the tutorials. Hence we selected 6 half-day tutorials and scheduled them on two days prior to the main conference, April 22 and 23. The program offers four tutorials that introduce the major and most-advanced technologies for AOSD, using material that has been at least once successfully presented at a previous ECOOP, OOPSLA or ICSE conference (T1, T2, T3 and T6). These tutorials require a moderate knowledge of OOSD and little or no knowledge of AOSD. Additionally the program includes two tutorials for those attendees who have a more advanced knowledge of AOSD (T4 and T5).

Monday
April 22 		T1 Advanced Software Composition: Obstacles & The Composition Filters Approach
T2 Hyper/J™: Multi-Dimensional Separation of Concerns for Java™
Tuesday
April 23 		T3 Introduction to Aspect-Oriented Programming with AspectJ
T4 Advanced Aspect-Oriented Programming with AspectJ
T5 AOP Language Exploration Using the Aspect Sand Box
T6 Aspect-Oriented Programming of Traversal-Related Concerns in Java

Note that we offer an Introductory Tutorial Package which consists of four tutorials - T1, T2, T3 and T6 - for the price of three.

Maja D'Hondt
AOSD 2002 Tutorials Chair
